# Student Tracker (Java SE CLI)

Консольное приложение для автоматизации учета успеваемости студентов, управления курсами и расчета академической аналитики.

## 🚀 Функционал
- [cite_start]**Управление данными:** Добавление студентов и учебных предметов в оперативную память приложения[cite: 8, 31, 32].
- [cite_start]**Учет успеваемости:** Выставление оценок с жесткой валидацией диапазона (от 0 до 100) и проверкой существования объектов[cite: 16, 22, 41, 42].
- **Академическая аналитика:**
  - [cite_start]Расчет индивидуального среднего балла (GPA) студента[cite: 23, 34].
  - [cite_start]Формирование выборки ТОП-3 студентов с наивысшей успеваемостью[cite: 27, 35].
  - [cite_start]Расчет среднего балла по каждому предмету[cite: 28, 36].
- [cite_start]**Надежность:** Полная обработка рантайм-исключений ввода (`NumberFormatException`) — система защищена от аварийного завершения[cite: 43].

## 📂 Архитектура проекта
[cite_start]Приложение спроектировано по принципу слоистой архитектуры (Layered Architecture)[cite: 47]:
- [cite_start]`model` — Доменные сущности (`Student`, `Course`, `Grade`) с инкапсулированными полями[cite: 13, 14, 15, 16, 17, 45].
- `service` — Слои бизнес-логики и аналитики. [cite_start]Оптимизация поиска по ID через хэш-таблицы (`HashMap` — $O(1)$)[cite: 18, 19, 25, 40, 45].
- [cite_start]`main` — Интерактивный CLI-интерфейс (Event Loop) для взаимодействия с пользователем[cite: 29, 30, 45].
